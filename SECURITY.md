# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in BitByteLabs, please report it privately.

- **Do not** open public issues for security problems.
- Email us at: security@bitbytelabs.com
- Include a detailed description and steps to reproduce the issue.

We will respond as quickly as possible and coordinate a fix.

## Supported Versions
We aim to support the latest major version. Please update to the latest release for security patches.

Thank you for helping keep BitByteLabs and its users safe!
